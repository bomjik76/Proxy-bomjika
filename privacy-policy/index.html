Privacy Policy

Browser Extension

info General Information

This Privacy Policy describes how the "Proxy bomjika" extension (hereinafter referred to as "Extension", "we", "us") processes information when using this browser extension.

Important: This extension does not collect, store, or use any user data. The extension only stores your proxy configuration settings locally on your device for its operation. No personal information, browsing history, or any other data is collected, stored, or transmitted.

storage Data Storage

The extension does not collect or store any user data. The only information stored locally on your device is the proxy configuration settings that you provide yourself:

  • Proxy server settings: address, port, username, and password (stored locally in the browser's storage for extension functionality only)
  • Extension state: enabled/disabled status and operating mode preferences (stored locally)
  • Domain list: list of websites you choose to add for proxy routing (stored locally)
lock
We do not collect, use, or transmit any of your data. All settings are stored exclusively in your browser's local storage (chrome.storage.local) and are used solely for the extension's functionality. No data is sent to external servers or third parties.

security Proxy Server Usage

When the extension is activated, your internet traffic may pass through the specified proxy server. Please note:

  • The extension uses the proxy server specified in the settings
  • We do not control and are not responsible for the operation of third-party proxy servers
  • It is recommended to use only trusted proxy servers
  • Your traffic passes through the proxy only for domains specified in the list (in selective mode)
Warning: When using a proxy server, your IP address may be visible to the proxy server operator. Make sure you trust the operator of the proxy server you are using.

visibility_off Data Collection Policy

The extension does NOT collect, store, or use any user data. We do not gather any information about you or your online activities.

The extension does NOT collect or transmit:

  • Your personal data (name, email, phone, etc.)
  • Web browsing history or visited websites
  • Passwords and authentication data
  • Payment information or financial data
  • Location data or device information
  • Any other information that could identify you or your activities
Zero Data Collection: This extension operates with a strict zero-data-collection policy. We do not track, monitor, analyze, or store any information about your browsing activities or personal data.

settings Extension Permissions

The extension requests the following permissions:

  • proxy: to configure the proxy server
  • storage: to save settings locally
  • webRequest: to intercept and route web requests
  • tabs: to get information about the current tab (when adding a site to the list)
  • <all_urls>: to apply proxy to specified domains
verified
All permissions are used exclusively for the extension's functionality and are not used for collecting, storing, or transmitting any user data. The extension does not analyze, track, or use any information about your browsing activities.

update Changes to Privacy Policy

We reserve the right to make changes to this Privacy Policy. We will notify users of significant changes through extension updates.

By continuing to use the extension after changes are made, you agree to the updated Privacy Policy.
